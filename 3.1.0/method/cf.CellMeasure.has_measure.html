cf.CellMeasure.has_measure¶

CellMeasure.has_measure()[source]¶

Whether the measure has been set.

New in version 1.7.0.

See also

del_measure, get_measure, set_measure

Returns
bool

True if the measure has been set, otherwise False.

Examples:

>>> c.set_measure('area')
>>> c.has_measure()
True
>>> c.get_measure()
'area'
>>> c.del_measure()
'area'
>>> c.has_measure()
False
>>> print(c.del_measure(None))
None
>>> print(c.get_measure(None))
None
