cf.Count.__data__¶

Count.__data__()[source]¶

Returns a new reference to the data.

Allows the construct to initialize a Data object.

Returns:Data

Examples:

>>> f.data
<CF Data(12): [14, ..., 56] km)
>>> cf.Data(f)
<CF Data(12): [14, ..., 56] km)
>>> cf.Data.asdata(f)
<CF Data(12): [14, ..., 56] km)
